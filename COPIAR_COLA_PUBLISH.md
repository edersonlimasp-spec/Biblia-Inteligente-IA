# 📋 COPIAR E COLAR - Pronto para o Publish UI

## 🎯 O Que Copiar e Onde Colar

### NO PUBLISH UI → Advanced Settings

---

## 1️⃣ BUILD COMMAND

**Campo que você vai ver:**
```
Build command *
┌─────────────────────────────────────┐
│ npm run build                       │  ← Vai estar isto ou vazio
└─────────────────────────────────────┘
```

**APAGUE tudo e COPIE E COLE isto:**
```
npm ci && npm run build
```

---

## 2️⃣ RUN COMMAND

**Campo que você vai ver:**
```
Run command *
┌─────────────────────────────────────┐
│ npm run start                       │  ← Deve estar isto
└─────────────────────────────────────┘
```

Se estiver diferente, APAGUE e COPIE E COLE isto:
```
npm run start
```

---

## ✅ Verificação

Depois que colar, verifique:

- [ ] Build command tem: `npm ci && npm run build`
- [ ] Run command tem: `npm run start`
- [ ] Sem espaços errados
- [ ] Sem typos
