Quero que você implemente no meu app nativo “Bíblia Hebraico & Grego – Primeiros Textos + IA” uma IA que NÃO use prompt solto, e sim um AGENTE criado na OpenAI (Professor Teológico).

A IA do app deve responder SEMPRE usando esse agente.

========================
1) ARQUITETURA DE IA
========================
- Criar um backend Node.js (Express ou NestJS) com endpoint /ai/ask
- Esse endpoint deve:
  1. Receber do app: pergunta do usuário + versículo atual (livro, cap, vers, texto) + idioma/versão.
  2. Chamar a OpenAI Responses API usando um AGENTE especificado pelo assistant_id.
  3. Retornar a resposta para o app.

- O app NUNCA deve chamar OpenAI direto no frontend.
- A chave API deve ficar apenas no backend (variável de ambiente).

========================
2) OPENAI AGENT
========================
- Usar a Responses API (plataforma Agents) com um assistant_id fixo.
- O assistant_id deve ser configurado via .env:
  OPENAI_API_KEY=xxxxx
  OPENAI_ASSISTANT_ID=asst_xxxxx

- A chamada para o agente deve incluir:
  a) system/base instructions já definidas no assistente (“Professor Teológico”).
  b) contexto do versículo atual enviado do app.
  c) pergunta do usuário.

========================
3) INTEGRAÇÃO NA TELA DA BÍBLIA
========================
- Na tela de leitura bíblica:
  • input fixo no rodapé (“Pergunte ao Professor Teológico”)
  • botão enviar
  • resposta aparece num painel acima do texto.
- Ao enviar, o app chama /ai/ask com:
  { pergunta, referencia, texto, versao }

========================
4) PLANO DE ACESSO À IA (novos preços)
========================
- Trocar os nomes e preços dos planos:

Plano GOLD – R$ 19,90 / mês
  • Acesso à IA Professor Teológico (modo Essencial)
  • Strong + Hebraico + Grego liberados enquanto a assinatura estiver ativa

Plano PREMIUM – R$ 29,90 / mês
  • Acesso à IA Professor Teológico em modo Premium (respostas profundas)
  • Strong + Hebraico + Grego liberados enquanto a assinatura estiver ativa

Strong Vitalício (sem IA) – R$ 189,90
  • Strong + Hebraico + Grego permanentes
  • sem IA

========================
5) TRIAL 30 DIAS
========================
- Todo usuário novo deve receber 30 dias de acesso TOTAL:
  • Strong
  • Hebraico/Grego
  • IA (modo Essencial)

- Após expirar:
  • bloquear Strong + IA automaticamente
  • Bíblia em português continua livre
  • ao tentar acessar IA ou Strong, redirecionar para tela de planos.

========================
6) CONTROLE DE CUSTO
========================
- Implementar rate-limit de segurança:
  • GOLD: até X perguntas/dia (ex.: 30)
  • PREMIUM: até Y perguntas/dia (ex.: 100)
- Mostrar ao usuário quando estiver perto do limite.

========================
7) ENTREGÁVEIS
========================
1. Código completo do endpoint /ai/ask
2. Exemplo de chamada usando assistant_id (Responses API)
3. Middleware de validação de plano (trial/Gold/Premium)
4. Ajuste na UI da tela bíblica para uso da IA
5. Testes:
   - usuário trial usa IA
   - trial expira → IA bloqueia
   - Gold desbloqueia
   - Premium desbloqueia
6. Revisar tudo antes de finalizar.

IMPORTANTE:
Antes de concluir, revise erros, inconsistências e partes faltando. Só finalize quando:
- IA estiver respondendo via o agent correto
- trial 30 dias funcionando
- planos Gold/Premium e Strong vitalício funcionando
- UI integrada na tela da Bíblia funcionando perfeita.
