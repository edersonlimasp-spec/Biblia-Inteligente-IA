## Contexto
Estou trabalhando no projeto “Bíblia Inteligente IA” no Replit. Dentro do módulo de leitura bíblica, há um campo de texto onde o usuário digita perguntas para o “Professor IA”. Atualmente esse input é pequeno e sem formatação refinada.

## Objetivo
Modificar esse campo para:
- **Ser maior e multiline**: o usuário deve poder digitar perguntas longas confortavelmente, com altura ajustável e quebra de linha (Shift+Enter para nova linha, Enter para enviar).
- **Estilo semelhante ao ChatGPT**: usar fonte limpa e moderna (ex.: `Inter`, `Helvetica Neue` ou sistema), espaçamento interno generoso (padding), bordas arredondadas, cor de fundo clara, contorno sutil e sombra discreta. A aparência deve lembrar o input do ChatGPT, mas sem copiar exatamente; adapte a paleta de cores do app (ex.: azul/cinza do tema) para manter identidade visual própria.
- **Acessibilidade**: contraste adequado, placeholder descritivo (“Digite sua pergunta para o Professor…”), e suporte a teclado (Tab foco).

## Tarefas técnicas
1. **HTML/JSX** – substitua o `<input>` atual por um `<textarea>` com `rows` configuráveis ou use um componente de UI (ex.: Chakra UI `Textarea`, Material UI `TextField multiline`).
2. **CSS/Estilo** – crie ou ajuste classes CSS (ou propriedades de estilo inline) para definir:
   - `min-height` maior (ex.: 60–80px), `width: 100%`.
   - `padding: 12px 16px`, `border-radius: 8px`, `border: 1px solid rgba(0,0,0,0.15)`.
   - `font-family: 'Inter', sans-serif` (ou equivalente).
   - `background-color` suave (ex.: `#F8F9FA`) e uma sombra leve para destacar o campo.
3. **Comportamento** – implemente o envio da pergunta ao pressionar Enter; use Shift+Enter para inserção de nova linha.
4. **Responsividade** – garanta que o campo se ajuste bem em dispositivos móveis e desktop, mantendo margens adequadas.

## Resultado esperado
Após as modificações, o campo de pergunta deve ocupar mais espaço horizontal/vertical, ser visualmente agradável e acolhedor como o do ChatGPT (mas com cores e detalhes próprios do aplicativo), permitindo digitação fluida e leitura confortável.
