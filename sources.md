## APIs Licenciadas Recomendadas

Para adicionar versões comerciais, considere estas APIs:

1. **API.Bible** (American Bible Society)
   - URL: https://scripture.api.bible/
   - Versões: 2500+ em 1600+ idiomas
   - Licença: Gratuita para uso não-comercial

2. **Bible.org NET Bible API**
   - URL: https://labs.bible.org/api_web_service
   - Versões: NET Bible
   - Licença: Gratuita com atribuição

3. **Biblia.com API** (Faithlife)
   - URL: https://api.biblia.com/
   - Versões: Várias (requer chave API)
   - Licença: Verificar termos

## Notas

- *NVI: Verificar status de licença específico para uso em apps
- Versões com "Licença Requerida" estão desabilitadas no dropdown até obter autorização
- Versões "Pendentes" são de domínio público mas precisam de dados importados
- O sistema usa fallback automático para versões sem dados (ex: AA → ACF)
