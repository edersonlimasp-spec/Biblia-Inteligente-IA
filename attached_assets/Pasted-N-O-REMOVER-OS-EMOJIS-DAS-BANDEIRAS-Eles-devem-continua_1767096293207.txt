PASSO 1 — CRIAR UMA FONTE ÚNICA DE VERDADE DO IDIOMA (GLOBAL + PERSISTENTE)
Crie src/i18n/languageStore.ts (ou .js):
- type Lang = 'pt' | 'en' | 'es'
- STORAGE_KEY = 'app_lang'
- getInitialLanguage(): Lang (localStorage -> fallback pt)
- persistLanguage(lang)
- React Context Provider (LanguageProvider) com:
  - state: lang
  - action: setLang(lang)
  - ao setLang: persistir + disparar i18n.changeLanguage(lang)
- hook useLanguage()

Se já houver Zustand/Redux, pode usar, mas com persistência e hook equivalente.
