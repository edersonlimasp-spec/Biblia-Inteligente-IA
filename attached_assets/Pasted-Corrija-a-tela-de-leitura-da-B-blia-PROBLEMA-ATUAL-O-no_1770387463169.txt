Corrija a tela de leitura da Bíblia.

PROBLEMA ATUAL:
O nome do livro/capítulo (ex: "Gênesis 1", "Salmos 110") está aparecendo no meio da tela de leitura e rola junto com os versículos.

OBJETIVO:
Quero que o nome do livro/capítulo fique CONGELADO no topo da tela de leitura, igual ao exemplo da segunda imagem (Salmos 110). O título deve ser grande, centralizado e permanecer fixo no topo enquanto os versículos rolam normalmente abaixo.

REQUISITOS OBRIGATÓRIOS:
1. O título do livro/capítulo deve ficar sempre visível no topo enquanto o usuário rola os versículos.
2. O título NÃO pode aparecer no meio do conteúdo.
3. O título deve ser um header separado do conteúdo, não pode estar dentro do bloco que rola.
4. O header deve usar position: sticky (preferencial) ou position: fixed.
5. Deve ter background sólido para não misturar com os textos ao rolar.
6. Deve ter z-index alto para ficar acima dos versículos.
7. O conteúdo deve ter espaçamento suficiente para não ficar escondido atrás do header.
8. O tamanho da fonte do título deve ser grande (igual ao exemplo da segunda imagem).
9. O título deve ficar centralizado.
10. Não pode haver duplicação do título.

IMPLEMENTAÇÃO:
- Localize o componente da tela de leitura (ex: BibleReader, ReadingView, ReaderPage, BibleChapterView ou similar).
- Remova o título do fluxo do conteúdo.
- Crie uma estrutura assim:

<div class="reading-wrapper">
  <div class="reading-title-sticky">Livro Capítulo</div>
  <div class="reading-content">versículos aqui...</div>
</div>

- O container "reading-content" deve ser o responsável pelo scroll, enquanto o título fica sticky no topo.

CSS OBRIGATÓRIO (adicione no arquivo global ou no CSS do componente):

.reading-title-sticky {
  position: sticky;
  top: 0;
  z-index: 9999;
  background: #ffffff;
  padding: 18px 16px;
  font-size: 32px;
  font-weight: 700;
  text-align: center;
  border-bottom: 1px solid rgba(0,0,0,0.08);
}

.dark .reading-title-sticky {
  background: #0b0f1a;
  border-bottom: 1px solid rgba(255,255,255,0.08);
}

.reading-content {
  padding-top: 8px;
}

Caso o projeto use container com overflow scroll, garanta que o sticky funcione corretamente e que o título fique preso no topo dentro da área de leitura.

VALIDAÇÃO FINAL:
- Ao abrir qualquer capítulo, o título aparece no topo (ex: "Gênesis 1", "Salmos 110") com letras grandes.
- Ao rolar para baixo, o título continua fixo no topo.
- Os versículos rolam abaixo normalmente.
- O título não aparece mais no meio do texto.
- O primeiro versículo não fica escondido atrás do header.
